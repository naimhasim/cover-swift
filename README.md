
# [Cover Swift](https://coverswift.naimhasim.com/)

## Development


```
git clone git@github.com:naimhasim/cover-swift.git

cp .env.example .env

touch database/database.sqlite

composer install & npm install

php artisan key:generate

npm run build

docker-compose up -d

php artisan migrate 

php artisan db:seed 

```
